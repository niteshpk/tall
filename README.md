
## About TALL

TALL Stack (Tailwind, Alpine.js, Laravel (8.*), and Livewire) Admin Dashboard

## Installation

``` bash
git clone https://github.com/takielias/tall.git
cd tall
composer install
npm install && npm run dev
```

## Run the server
```bash
php artisan serve
```
